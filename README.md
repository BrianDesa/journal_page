# Ceylon Journal of Science - Landing Page Template

A professional, responsive landing page for the **Ceylon Journal of Science**, University of Peradeniya. This project serves as a modern web portal for academic publishing, featuring smooth navigation and an interactive editorial board.

## 🌟 Key Features

* **Glassmorphism UI:** A sticky navigation bar with backdrop-blur effects.
* **Responsive Design:** Optimized for mobile, tablet, and desktop views using CSS Media Queries.
* **Interactive Editorial Carousel:** A custom JavaScript-powered horizontal scroll for board members.
* **Modern Typography:** Elegant font pairing using *Merriweather* (Serif) and *Source Sans 3* (Sans-Serif).
* **SEO Friendly:** Uses semantic HTML5 tags for better indexing.

## 🛠️ Built With

* **HTML5** - Document structure.
* **CSS3** - Custom styling, Flexbox, and CSS Variables.
* **JavaScript (Vanilla)** - Toggle menu and scroll logic.
* **Google Fonts** - Merriweather & Source Sans 3.
